# Spring Market Application

ADD to Application.properties:

- DB_USERNAME = {YOUR_DB}

- DB_PASSWORD = {YOUR_PASSWORD}

- DB_URL = {YOUR_DB_URL}
#For example:

* DB_USERNAME = root

* DB_PASSWORD = super_password

* DB_URL = jdbc:postgresql://localhost:5432/MYSQL_DB
